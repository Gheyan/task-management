Follow these steps to run the project locally after cloning.

## Install Node.js

Download and install Node.js (LTS version):
https://nodejs.org/

Verify installation:
node -v
npm -v

## Install Visual Studio Code (VS Code)

Download and install VS Code:
https://code.visualstudio.com/

(Optional) Verify by opening a terminal and running:
code --version

## Clone the Repository

git clone https://github.com/Gheyan/task-management.git

## Open Project

cd .\task-management\task-management\
code .

## Install Dependencies

Open the terminal within the code editor and type: 
npm install

## Setup Environment Variables

Place the necessary .env file within the root directory:
.\task-management\task-management\

## Run the Project

Open the terminal within the code editor and type: 
npm run dev

Open in browser:
http://localhost:5173
